# Roblox Tools 

This repo will have various tools to improve quality of life on roblox

Current tools:
- Roblox AFK (Requires [AutoHotkey 2.0](https://www.autohotkey.com/download/ahk-v2.exe))

##### Let's you do all your stuff while making sure you don't get kicked on your account(s)

Planned tools:
- Multi roblox

##### Let's you launch multiple accounts, very basic, you can even minimize it into the tray!
